# 梅花心易占卜应用 - 产品评估指标框架 (Metrics Framework)

## 1. 指标框架概述

### 1.1 指标体系目的
建立科学的产品评估指标体系，通过数据驱动产品决策，确保产品朝着正确方向发展，实现用户价值和商业价值的平衡。

### 1.2 指标设计原则
- **可衡量性**：所有指标都可以准确测量和追踪
- **可操作性**：指标变化能够指导具体的产品优化行动
- **业务相关性**：指标直接关联核心业务目标
- **平衡性**：兼顾短期和长期、用户价值和商业价值

### 1.3 指标分层结构
```
北极星指标 (1个)
├── 关键结果指标 KRs (3-5个)
├── 驱动指标 (5-8个)
└── 监控指标 (10-15个)
```

## 2. 北极星指标定义

### 2.1 北极星指标选择
**指标名称**：月度付费占卜次数 (Monthly Paid Divination Count)

### 2.2 选择依据
- **用户价值体现**：用户愿意付费说明产品提供了真实价值
- **商业价值直接**：直接关联收入和商业成功
- **行为导向明确**：指导团队专注于提升占卜质量和用户体验
- **可持续增长**：反映产品的长期健康发展

### 2.3 目标设定
- **MVP阶段**：月度付费占卜次数 > 100次
- **Beta阶段**：月度付费占卜次数 > 500次
- **正式版**：月度付费占卜次数 > 2000次
- **成熟期**：月度付费占卜次数 > 10000次

### 2.4 计算公式
```
月度付费占卜次数 = Σ(当月所有用户的付费占卜次数)
不包括免费次数和邀请奖励次数
```

## 3. HEART指标体系详述

### 3.1 Happiness (用户满意度)

#### 3.1.1 核心指标
**用户满意度评分 (User Satisfaction Score)**
- **定义**：用户对占卜结果和整体体验的满意度评分
- **计算方式**：5分制评分的平均值
- **目标值**：≥ 4.0分
- **收集方式**：占卜完成后弹窗评分、定期用户调研

#### 3.1.2 辅助指标
- **AI解读有用性评分**：用户对AI解读内容的有用性评价
- **产品推荐意愿 (NPS)**：用户向朋友推荐产品的意愿度
- **用户投诉率**：用户投诉数量占总用户数的比例

### 3.2 Engagement (用户参与度)

#### 3.2.1 核心指标
**日活跃用户数 (DAU)**
- **定义**：每日至少进行一次占卜的用户数量
- **目标值**：月增长率 > 20%
- **计算周期**：每日统计，按周/月分析趋势

**用户会话时长**
- **定义**：用户单次访问的平均停留时间
- **目标值**：≥ 5分钟
- **关键页面**：占卜流程页面、结果查看页面

#### 3.2.2 辅助指标
- **页面浏览深度**：用户平均访问页面数量
- **功能使用率**：各功能模块的使用频率
- **用户互动率**：分享、评论等互动行为频率

### 3.3 Adoption (功能采用度)

#### 3.3.1 核心指标
**新用户激活率 (New User Activation Rate)**
- **定义**：注册后7天内完成首次占卜的用户比例
- **计算公式**：7天内完成首次占卜用户数 / 新注册用户数
- **目标值**：≥ 80%

**核心功能使用率**
- **占卜功能使用率**：注册用户中使用过占卜功能的比例
- **历史记录查看率**：用户查看历史占卜记录的比例
- **分享功能使用率**：用户分享占卜结果的比例

#### 3.3.2 辅助指标
- **功能发现率**：用户发现各功能的比例
- **功能完成率**：用户完整使用功能流程的比例
- **新功能采用速度**：新功能发布后的用户采用时间

### 3.4 Retention (用户留存)

#### 3.4.1 核心指标
**用户留存率 (User Retention Rate)**
- **次日留存率**：注册后第2天仍活跃的用户比例，目标 ≥ 40%
- **7日留存率**：注册后第7天仍活跃的用户比例，目标 ≥ 30%
- **30日留存率**：注册后第30天仍活跃的用户比例，目标 ≥ 20%

**付费用户留存率**
- **付费用户月留存**：付费用户在次月继续使用的比例
- **会员续费率**：会员到期后续费的比例

#### 3.4.2 辅助指标
- **用户回访频率**：用户平均多久回访一次
- **流失用户分析**：流失用户的特征和流失原因
- **重新激活率**：流失用户重新使用产品的比例

### 3.5 Task Success (任务完成度)

#### 3.5.1 核心指标
**占卜成功完成率**
- **定义**：用户发起占卜到获得结果的成功率
- **计算公式**：成功完成占卜次数 / 发起占卜次数
- **目标值**：≥ 99%

**支付成功率**
- **定义**：用户发起支付到支付成功的比例
- **目标值**：≥ 98%

#### 3.5.2 辅助指标
- **注册完成率**：开始注册到完成注册的比例
- **问题提交成功率**：用户提交问题的成功率
- **页面加载成功率**：页面正常加载的比例

## 4. AARRR漏斗指标详述

### 4.1 Acquisition (获客)

#### 4.1.1 核心指标
- **新用户注册数**：每日/周/月新注册用户数量
- **获客成本 (CAC)**：获得一个新用户的平均成本
- **获客渠道效果**：不同渠道的用户质量和转化率

#### 4.1.2 渠道分析
- **自然流量**：搜索引擎、直接访问
- **推荐流量**：用户邀请、口碑传播
- **付费流量**：广告投放、合作推广

### 4.2 Activation (激活)

#### 4.2.1 核心指标
- **用户激活率**：完成关键行为的新用户比例
- **首次占卜完成率**：新用户完成首次占卜的比例
- **激活时间**：从注册到激活的平均时间

#### 4.2.2 激活定义
用户激活 = 注册 + 完善资料 + 完成首次占卜

### 4.3 Retention (留存)
参考HEART框架中的留存指标

### 4.4 Revenue (收入)

#### 4.4.1 核心指标
**月度经常性收入 (MRR)**
- **定义**：每月来自用户付费的收入总额
- **目标值**：
  - Beta版：MRR > 1000元
  - 正式版：MRR > 5000元
  - 成熟期：MRR > 20000元

**用户平均收入 (ARPU)**
- **定义**：每个用户的平均收入贡献
- **计算公式**：总收入 / 总用户数
- **目标值**：≥ 10元/用户/月

#### 4.4.2 辅助指标
- **付费转化率**：免费用户转为付费用户的比例
- **客户生命周期价值 (LTV)**：用户在整个生命周期的价值
- **收入增长率**：月度/季度收入增长比例

### 4.5 Referral (推荐)

#### 4.5.1 核心指标
- **邀请转化率**：邀请链接转化为注册的比例
- **病毒系数 (K-factor)**：每个用户平均带来的新用户数
- **推荐用户质量**：推荐用户的留存和付费表现

#### 4.5.2 推荐机制效果
- **邀请发送率**：用户发送邀请的比例
- **邀请接受率**：收到邀请的用户注册比例
- **推荐用户LTV**：推荐用户的生命周期价值

## 5. 功能级评估指标

### 5.1 占卜核心功能

#### 5.1.1 使用指标
- **占卜发起次数**：用户发起占卜的总次数
- **占卜完成率**：成功完成占卜的比例
- **平均占卜时长**：从提问到查看结果的时间

#### 5.1.2 质量指标
- **AI解读生成成功率**：AI成功生成解读的比例
- **AI解读生成时间**：平均生成时间
- **用户对解读的满意度**：解读质量评分

#### 5.1.3 业务指标
- **免费占卜使用率**：免费次数的使用情况
- **付费占卜转化率**：免费用完后的付费转化
- **重复占卜率**：用户进行多次占卜的比例

### 5.2 用户管理功能

#### 5.2.1 注册登录
- **注册转化率**：访问到注册完成的转化率
- **注册完成时间**：平均注册流程耗时
- **登录成功率**：用户登录的成功比例

#### 5.2.2 用户资料
- **资料完善率**：用户完善个人资料的比例
- **资料准确性**：用户资料的真实性和完整性

### 5.3 支付功能

#### 5.3.1 支付转化
- **支付页面到达率**：用户到达支付页面的比例
- **支付完成率**：发起支付到完成支付的比例
- **支付失败率**：支付失败的比例和原因

#### 5.3.2 会员系统
- **会员转化率**：普通用户转为会员的比例
- **会员续费率**：会员到期后续费的比例
- **会员活跃度**：会员用户的使用频率
